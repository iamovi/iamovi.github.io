Hey there! I'm Ovi ren, a developer who builds things on the internet. Some of them are useful, some are just fun, and some make people question my design choices. I'm cool with all three.

Started coding as a hobby and somehow ended up with a GitHub full of projects. Funny how that happens. What began as "let me try this" turned into "oh wait, this is actually awesome" and here we are. Also, I'm an overthinker 💭 — which is just a fancy way of saying I spend way too much time thinking about edge cases and whether my variable names make sense.

What I Actually Do

I'm a hobby programmer specializing in Node.js and Python, but I'm also a language explorer — always trying out new tech and languages just to see what they can do. I build everything from backend APIs to frontend experiments, and I'm not afraid to throw in some anime aesthetics because, honestly, why should code be boring?

My projects range from practical tools to creative experiments. If it's interesting to build, I'm probably going to build it. The internet has enough boring websites already — I'm here to add a little personality (and sarcasm) to the mix.

Open source enthusiast, documentation reader extraordinaire, sarcastic guy who believes that good code should be both functional and fun. Also, dark mode for everything. There's no debate here.
